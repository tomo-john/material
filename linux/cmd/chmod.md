### chmod
change mode
ファイルやディレクトリの権限を変更する。
- 4 => read : 読み取り
- 2 => write : 書き込み
- 1 => execute : 実行

各桁の数値はこれらの値の合計

```
# ls -l した時に左側に出るやつ
-rwxrw-r--
# (左から)
-   : ファイルかディレクトリか(ディレクトリならd)
rwx : ユーザーの権限(これは7)
rw- : グループの権限(これは6)
r-- : その他ユーザーの権限(これは4)
# 例
chmod 764 'filename'
=> -rwxrw-r-- になる
```

数値以外でのアクセス権変更操作
```
# 操作対象
u : 所有者
g : 所有グループ
o : その他のユーザー

# 操作
+ : 権限を追加
- : 権限を削除
= : 権限を指定

# アクセス権
r : 読み取り
w : 書き込み
x : 実行

# 例
chmod u+x file
=> 所有者に実行権限を追加

chmod go-wx file
=> 所有グループ、その他のユーザーから書き込み権限と実行権限を削除
```
