# Font Awesome

## HTMLにリンク

`<head>`タグに以下のリンクを埋め込む。

```
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.2/css/all.min.css">
```

[Font Awesome](https://fontawesome.com/icons)にあるクラスを指定するだけでアイコンが使える :dog:

例: `<i class="fa-solid fa-dog icon-dog"></i>`

## 組み込みサイズクラス

| クラス名 | 意味           |
|----------|----------------|
| `fa-xs`  | 特小           |
| `fa-sm`  | 小さめ         |
| `fa-lg`  | 大きめ(1.25倍) |
| `fa-2x`  | 2倍            |
| `fa-3x`  | 3倍            |
| `fa-5x`  | 5倍            |
| `fa-10x` | 10倍           |
