# 2進数の除算

除数(割る数)が2の累乗(2, 4, 8, 16, ...)の場合:

- 除数の1よりも右側の部分に対応する被除数(割られる数)の部分があまり
- 残りが商

### 例1: 170 / 4 = 42 あまり 2

```
10101010 (170)
00000100 (4)

4は3bit目に1がある。
10101010の3bitより右側に対応する10(2)があまり。
残りの101010(42)が商。
```

### 例2: 85 / 8 = 10 あまり5

```
01010101 (85)
00001000 (8)

=> 4ビットより右が 101(5)であまり。
=> 残りの01010(10)が商。
```
