# HTML要素の2つの基本タイプ

HTML要素には`ブロック要素`と`インライン要素`の2つの基本的な性質に分類される。

## block

`display: block`

ブロック要素は縦の領域を占有する要素。

| 特徴   | 詳細                                                               |
|--------|--------------------------------------------------------------------|
| 幅     | 何も指定しない限り、親要素の横幅いっぱい(100%)に広がる             |
| 配置   | 前後に他の要素があっても、必ず改行して別の行から始まる             |
| 余白   | `width`, `height`, `margin`, `padding`などすべてのサイズ指定が可能 |
| 代表例 | `div`, `p`, `h1～h6`, `ul`, `li`                                   |

## inline

`display: inline`

インライン要素はテキストの一部として扱われる要素。

| 特徴   | 詳細                                                                               |
|--------|------------------------------------------------------------------------------------|
| 幅     | 中身のコンテンツの分だけしか広がらない(最小限の幅)                                 |
| 配置   | 前後の要素と同じ行に並んで表示させ、改行されない                                   |
| 余白   | `margin`, `padding`は左右のみ有効で、上下の`margin`や`height`, `width`は無視される |
| 代表例 | `a`, `span`, `strong`, `em`, `img`                                                 |

## inline-block

`display: inline-block`

インライン要素とブロック要素のそれぞれの性質を持つ。

インラインの性質を残すので、他の要素と同じ行に並ぶ。(改行しない)

ブロックの性質を取り入れるので、`width`, `height`などのサイズ指定がすべて有効になる。
