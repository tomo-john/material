## 正方行列

行(横)と列(縦)の数が同じ行列のこと。

```
# 3行3列の正方行列(3x3)

A = | 1 2 3 |
    | 4 5 6 |
    | 7 8 9 |
```

- 正方行列でないと逆行列は定義できない
- 単位行列も正方行列の1種

## 単位行列

対角成分がすべて1、その他が0の正方行列。

```
I = | 1 0 0 |
    | 0 1 0 |
    | 0 0 1 |
```

- 掛け算の単位元
- `A x I = A`, `I x A = A`

## 逆行列

行列Aに対して、ある行列Bが存在し、`A x B = B x A = I`となるとき、BをAの逆行列と呼ぶ。

=> `B = A^-1`(Aの-1乗)と表す

```
A = | a b |
    | c d |
```

この場合、Aが逆行列をもつ条件は、`ad - bc ≠ 0`となる。

```
# 逆行列の公式
A^-1 = (1 / (ad - bc)) x | d -b |
                         | -c a |
```
