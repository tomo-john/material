# HTMLエンティティ

[MDNリンク](https://developer.mozilla.org/ja/docs/Glossary/Entity)

HTMLで特別な意味を持つ文字(`<`や`>`など)やキーボードで直接打ちにくい記号をブラウザに正しく表示させるための合言葉みたいなもの。

`&`で始まり、`;`で終わる形式で記述する。

## 基本の4つ

HTMLタグとして認識されてしまうのを防ぐために使用される。

| 記号 | エンティティ | 意味                              |
|------|--------------|-----------------------------------|
| `<`  | `&lt;`       | less than(左山括弧)               |
| `>`  | `&gt;`       | greater than(右山括弧)            |
| `&`  | `&amp;`      | ampersand(アンド)                 |
| `"`  | `&quot;`     | quotation(ダブルクォーテーション) |

## デザイン・文章でよく使う記号

| 記号  | エンティティ | 意味                              |
|-------|--------------|-----------------------------------|
| `©`   | `&copy;`     | Copyright(著作権)                 |
| `»`   | `&raquo;`    | Right pointing quote(次への矢印)  |
| `«`   | `&laquo;`    | Left pointing quote(前への矢印)   |
| `x`   | `&times;`    | かける・閉じるボタン(x)           |
| `...` | `&hellip;`   | 3点リーダー(三点)                 |
