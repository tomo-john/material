# CSSでの色の指定

ディスプレイ上の色はRed(赤)、Green(緑)、Blue(青)の3色の組み合わせで表現ができる。

CSSではこの3色の組み合わせ(RGB)を数値に置き換えて色を指定する。

透明度(Alpha channel)を表現することも可能。

### カラーコードの指定

- 16進数で指定
- 先頭に`#(ハッシュ)`をつける
- `#64bdb2`みたいな感じ

### RGBA(透明度あり)の場合

- 10進数で指定(0~255)
- RGBAの各値を`,(カンマ)`で区切る
- 透明度は0.7なら70%という意味
- `rgba(100,189,178,0.7)`みたいな感じ

どちらの指定方法でもOK。

透明度を指定したいときはRGBAを使用する。

`pink`のように色の名前で指定することも可能。
