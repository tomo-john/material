# 枠線

## border

```
border: 3px solid #000000;
```

これはショートハンドの書き方。

ショートハンドはいくつかのCSSプロパティで使用できる。

上記の場合:

- border-width: 3px;
- border-style: solid;
- border-color: #000000;

の3つをまとめたもの。

### border-styleの値

- solid: 実戦
- double: 二重線
- dotted: 点線
- dashed: 破線

### borderの4辺

任意の辺のborderを個別に指定することもできる。

=> border-top, border-bottom, border-left, border-right

## border-radius

ボックスや画像の四隅の丸みを一括で指定する。

値には単位を伴う数値が入る。
