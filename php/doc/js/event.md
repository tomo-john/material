# $eventとは何者？🐶

Alpine.jsにおいて`$event`は、ブラウザが標準で持っている`ネイティブのEventオブジェクト`を参照するためのマジック変数。

例えば、ボタンをクリックした時に`$event`の中身を除くと、クリックされた場所などのそのイベントに関する膨大な情報が詰まっている。

## $event.offsetX / $event.offsetY

イベントが発生した要素(ターゲット)内での水平(X)、垂直(Y)の距離。

`@click="fun($event)"`みたいに関数に`$event`を渡し、関数の中で`$event.offset(X)`が使用できる。

offset以外にも座標を示すプロパティはいくつかある。(ややこしいので注意)

| プロパティ名 | 基準となる場所                                  |
| ------------ | ----------------------------------------------- |
| offsetX / Y  | クリックした要素の左上が (0, 0)                 |
| clientX / Y  | ブラウザの表示領域(ビューポート)の左上が (0, 0) |
| pageX / Y    | ページ全体の左上が (0, 0)(スクロール分も含)     |
