# mathematics (算数)

## 割り算

```
32 ÷ 10 = 32 / 10 = 商が3...余りが2

10 ÷ 5 = 10 / 5 = 商が2...余りが0
```

## 分数

```
a = b / c => c = b / a

a = b / (c x d) => c = b / (a x d)
```
