# tr

trコマンドはファイル内の文字を削除・置換するコマンド。

```
tr [オプション] [文字列1 [文字列2]]
```

| オプション | 説明                                             |
|------------|--------------------------------------------------|
| -d         | 文字列1で指定した文字を削除                      |
| -s         | 文字列1で指定した文字が連続した場合、1文字に置換 |


trコマンドは変換対象を標準出力からのみ受け付ける。

変換・削除したいファイルをコマンドの引数に指定することができない。

```
# file1内の連続した#を#1文字に置換してfile2に出力
tr -s '#' < file1 > file2
```
