# ファイルテスト演算子

| 演算子 | 意味                                 |
|--------|--------------------------------------|
| -e     | 存在するか                           |
| -f     | 通常ファイルか(ディレクトリではない) |
| -d     | ディレクトリか                       |
| -r     | 読み込み可能か                       |
| -w     | 書き込み可能か                       |
| -x     | 実行可能か                           |
| -s     | サイズが0より大きいか(中身があるか)  |
