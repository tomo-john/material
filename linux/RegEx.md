## Regular expression (RegEx)
正規表現

### 正規表現の主なメタキャラクタ
```
# シェルのメタキャラクタとは異なるので注意

.  : 任意の1文字にマッチ
*  : 直前にある文字の0回以上の繰り返しにマッチ
?  : 直前にある文字の0回、または1回以上の繰り返しにマッチ(grep -E使用)
+  : 直前にある文字の1回以上の繰り返しにマッチ(grep -E使用)
[] : []内の任意の1文字にマッチ
^  : 行頭にマッチ(最初にあるときのみ有効)
$  : 行末にマッチ(最後にあるときのみ有効)
\  : 直後に続くメタキャラクタを打ち消す


# 使用例

a.c        : abc, a1cなｄはマッチ / Abc, acなどにはマッチしない
.*         : 0文字以上の文字列にマッチ
[A-Z]      : 大文字のアルファベット1文字にマッチ
[0-9][0-9] : 2桁の数字にマッチ
^$         : 空行(改行のみの行)にマッチ
^#         : 行頭の#文字にマッチ(行の中はマッチしない)
\.txt      : 「.txt」文字列にマッチ(.を\でエスケープしている)
friends?   : friend, friendsにマッチ
123+       : 123, 1233などにはマッチ / 12などにはマッチしない
```

### grep
テキストファイルの内容を検索し、指定した正規表現にマッチする行があれば表示する
```
grep [オプション]　[文字列パターン] [検索対象ファイル]

# オプション
-F : 文字列パターンを正規表現でなく単なる文字列として扱う
-c : マッチした行数だけを表示する
-i : 大文字と小文字を区別しない
-l : マッチした行のあるファイル名のみを表示する
-n : 行番号も合わせて表示
-v : マッチしなかった行を表示

# 使用例

grep www /etc/services
=> /etc/servicesファイルから「www」という文字列が含まれる行を表示

grep bash /etc/default/*
=> /etc/defaultディレクトリ以下のファイルから「bash」が含まれる行を表示

ps aux | grep bash
=> psの実行結果から「bash」が含まれる行だけを表示
```
