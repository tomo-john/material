# View

## Viewの命名規則

`view()`ヘルパーが使うドット記法は、ファイルシステム上のディレクトリの区切りを書き換えるためのもの。

| 指定方法          | 実際のファイルパス                           |
|-------------------|----------------------------------------------|
| `welcome`         | `resources/views/welcome.blade.php`          |
| `dogs.index`      | `resources/views/dogs/index.blade.php`       |
| `admin.user.list` | `resources/views/admin/users/list.blade.php` |
