# cron

`crond`デーモンによって指定されたスケジュールに基づいてコマンドを実行する

=> 通常は`crontab`というファイルに書かれる

## crontabファイルの構成
```
# 分 時 日 月 曜日 コマンド
  * * * * * cmd

分       : 0-59
時       : 0-23
日       : 1-31
月       : 1-12
曜日     : 0-7 => 0が日曜日
コマンド : フルパスで指定
```
## crontabの管理
```
crontab -l : 現在の内容を表示
crontab -e : 内容の編集
crontab -r : 削除(注意が必要)
```
