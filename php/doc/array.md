# 配列

複数のデータ(値)を順番にしまっておける大きな箱。

## 配列の作り方

PHPでは、`[](角カッコ)`を使用して配列を作る。

```
<?php
// fruits という名前の配列を作る
// 3つの文字列(データ)を順番に入れている
$fruits = ["りんご", "みかん", "ばなな"];
```

## 配列の中身の取り出し方

インデックス(添字)という番号を使用。

`$変数名[番号];`のように書く。

インデックスの番号は1からではなく、`0`から数え始めるのがルール。

```
<?php
$fruits = ["りんご", "みかん", "ばなな"];
echo $fruits[0] . "と" . $fruits[2] . "</br>";
```

## 配列にデータを追加する

`$変数名[] = データ;`のように書くと、配列の一番最後に新しいデータを追加してくれる。

```
$fruits = ["りんご", "みかん", "ばなな"];
$fruits[] = "いちご"; // 末尾に追加される
```

## 連想配列

配列には、インデックス番号の代わりに、自分で好きな名前(キー)を付けることもできる。

これを連想配列という。(Laravelでよく使う。超重要。)

`"キー" => "値"`というペアで書いていく。

```
$user = [
  "name" => "じょん",
  "age" => 2,
  "job" => "犬"
];

echo "名前: " . $user["name"] . " / 年齢: " . $user["age"] .  " / 職業: " . $user["job"] . "</br>";
```

- 取り出す時は、インデックス番号の代わりに`キーの名前を使う`(クォーテーションいる)
- `$user[0]`のような番号では取り出せない
