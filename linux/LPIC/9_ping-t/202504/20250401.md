# ランレベル

- 0: システム停止
- 1: シングルユーザーモード
- 2: マルチユーザーモード(テキストログイン、NFSなし)
- 3: マルチユーザーモード(テキストログイン)
- 4: 未使用
- 5: マルチユーザーモード(GUIログイン)
- 6: 再起動

=> Debian系の場合、ランレベル2～5を区別しない

# nl

指定したファイルに行番号をつけて出力するコマンド。

- nl -b a: 空行を`含めた`全ての行に行番号をつける

  => cat -n と同じ

- nl -b t: 空行を`除いた`行に行番号をつける(デフォルト)

  => cat -b と同じ

`cat -n = nl -b a`

`cat -b = nl -b t(nl)
