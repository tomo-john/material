# Linux

## プロンプト

```
# 引数
$_ : 直前のコマンドの最後の引数
!! : 直前のコマンド全体の再実行
=> !!:2 で直前のコマンドの2番目の引数を指定もできる

# カーソル移動
ctrl + A : 左端へ
ctrl + E : 右端へ

# 削除系
ctrl + U : カーソルから行先頭まで削除
ctrl + K : カーソルから行終端まで削除
ctrl + W : カーソル左の単語を削除

```

## プロンプトの定義
環境変数`PS1`によってプロンプトを定義している

- ~/.zshrc(MAC)
```
PS1='%n@%F{magenta}[%1~]%f$ '
```

|パラメータ| 内容                            |
|----------|---------------------------------|
| %n       | ユーザー名                      |
| @        | @という文字列(意味はない)       |
| %F{}     | 色の表示変更開始 {}内で色を指定 |
| magenta  | magenta色を指定している         |
| %1~      | 現在いるディレクトリ            |
| []       | []という文字列(意味はない)      |
| %f       | 色の表示変更終了                |
| $        | $という文字列(意味はない)       |

## find & locate

| 特徴     | find                         | locate                             |
|----------|------------------------------|------------------------------------|
| 検索方法 | リアルタイムにファイルを検索 | 事前に作成したデータベースから検索 |
| 検索速度 | 遅い(特に広範囲の場合)       | 高速                               |
| 更新情報 | 常に最新                     | データベース依存                   |
| 詳細条件 | 指定可能                     | 指定不可                           |
