# サンプルプログラム

```
#!/bin/bash

age=$1
ret=0

if [ "$age" -le 3 ]; then
  ret=100
elif [ "$age" -le 9 ]; then
  ret=300
else
  ret=500
fi

echo "$ret"
```

- ageが0から3の場合、retは100
- ageが4から9の場合、retは300
- ageが10以上の場合、retは500

# 二次元配列

要素が行と列の2方向に並ぶ配列。

```

 {0,2,2,0,0},
 {0,0,0,1,3},
 {0,0,0,2,0},
 {0,0,0,0,1}}
```

# プログラム記憶方式

実行に必要なプログラムを主記憶上に配置してから始める。

主記憶に配置されたプログラムの命令をCPUが順番に取り出して解読・実行する方式をプログラム記憶方式(プログラム格納方式)という。

主記憶はCPUが直接アクセスできる記憶装置。(メモリ・メインメモリ)
