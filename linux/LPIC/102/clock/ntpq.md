# ntpq

NTPデーモン(`ntpd`)と対話的に通信して、状態や統計情報を取得するツール。

主に時刻同期が正しく行われているかを確認するために使う。

| オプション | 説明                                                                          |
|------------|-------------------------------------------------------------------------------|
| -p         | serverに指定したサーバーとの同期状態を一覧表示する(対話モードの`peers`と同じ) |
| -i         | 対話モードで起動(デフォルト)                                                  |
