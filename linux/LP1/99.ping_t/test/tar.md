# tar

| オプション | 意味                           |
|------------|--------------------------------|
| -f         | アーカイブファイルの名前を指定 |
| -v         | アーカイブ処理の詳細を表示     |
| -c         | アーカイブの作成               |
| -x         | アーカイブの展開               |
| -t         | アーカイブの内容を表示         |
| -z         | gz形式(gzip)で圧縮・展開       |
| -j         | bz2形式(bzip2)で圧縮・展開     |
| -J         | xz形式で圧縮・展開             |
