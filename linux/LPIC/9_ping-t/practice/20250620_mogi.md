# routeコマンド

```
route del default gw 192.168.1.254
```

- -host : 対象をホストのアドレスとみなす
- -net : 対象をネットワークアドレスとみなす
- netmask : サブネットマスクを指定
- gw : ゲートウェイを指定
- default gw : デフォルトゲートウェイを指定

IPアドレスを指定せずに`route del default`でもデフォルトゲートウェイの削除は可能。(その場合`gw`は付けない)
