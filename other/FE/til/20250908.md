# SMTP-AUTH

メール投稿にあたってユーザー認証の仕組みがないSMTPを拡張し、ユーザー認証機能を追加した仕様。

利用するにはメールサーバーとクライアントの双方が対応していなければならない。

メール送信する際にユーザー名とパスワードで認証を行い、認証されたユーザーからのみからメール送信を許可することで不正な送信要求を遮断。

# SWOT分析

- S : Strength(強み)
- W : Weakness(弱み)
- O : Opportunity(機会)
- T : Threat(脅威)

強みと弱みは内部要因、機会と脅威は外部要因。
