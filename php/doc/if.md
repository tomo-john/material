# if文

if文は他プログラミング言語同様に、制御構文の1つ。

## ifの基本形

```
<?php
$score = 85;
echo "得点は" . $score . "点です。</br>";

if ($score >= 85) {
  echo "85点以上です</br>";
}
```

`if (条件文)`で始まり、条件を満たすときに`{ ... }`内の処理を行う。

## else

基本のif文だけの場合、条件を満たさなかったときは何も処理を行わない。

`else`は、条件を満たさない場合に実行したい処理を記述する。

```
$score = 80;
echo "得点は" . $score . "点です。</br>";

if ($score >= 85) {
  echo "85点以上です。</br>";
} else {
  echo "85点未満です。</br>";
}
```

## elseif

if文の条件をさらに細かく分ける。

`if`と`else`の間にいくつでも追加できる。

```
$num = 101;

if ($num % 15 == 0) {
  echo $num . "は15で割り切れます。</br>";
} elseif ($num % 5 == 0) {
  echo $num . "は5で割り切れます。</br>";
} elseif ($num % 3 == 0) {
  echo $num . "は3で割り切れます。</br>";
} else {
  echo $num . "は3, 5, 15では割り切れません。</br>";
}
```
