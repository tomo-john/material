## 進数対応表(8ビット：0〜255)

| 2進数       | 16進数 | 8進数  | 10進数 |
|-------------|--------|--------|--------|
| 0000 0000   | 00     | 000    | 0      |
| 0000 0001   | 01     | 001    | 1      |
| 0000 0010   | 02     | 002    | 2      |
| 0000 0011   | 03     | 003    | 3      |
| 0000 0100   | 04     | 004    | 4      |
| 0000 0101   | 05     | 005    | 5      |
| 0000 0110   | 06     | 006    | 6      |
| 0000 0111   | 07     | 007    | 7      |
| 0000 1000   | 08     | 010    | 8      |
| 0000 1001   | 09     | 011    | 9      |
| 0000 1010   | 0A     | 012    | 10     |
| 0000 1011   | 0B     | 013    | 11     |
| 0000 1100   | 0C     | 014    | 12     |
| 0000 1101   | 0D     | 015    | 13     |
| 0000 1110   | 0E     | 016    | 14     |
| 0000 1111   | 0F     | 017    | 15     |
| 0001 0000   | 10     | 020    | 16     |
| ...         | ...    | ...    | ...    |
| 1111 1111   | FF     | 377    | 255    |

- 2進数 => 16進数：4ビットずつに区切って読む
  - 例：`10111100` => `1011 1100` => `B C`

- 2進数 => 8進数：3ビットずつに区切って読む
  - 例：`101110` => `101 110` => `5 6`

- 16進数1桁 => 4ビット
- 16進数2桁 => 8ビット
- 8ビット => 1バイト=> 16進数なら2桁、8進数なら最大3桁
