# リダイレクト

| 記号 | 意味                                                   |
|------|--------------------------------------------------------|
| <    | 標準出力へ送る                                         |
| <<   | 終了文字までを標準入力へ送る                           |
| >    | 標準出力をファイルへ送る                               |
| >>   | 標準出力をファイルへ追記                               |
| 2>   | 標準エラー出力の出力先を指定                           |
| 2>&1 | 標準エラー出力の出力先を、標準出力の出力先と一緒にする |
