# 8進数
8進数(Octal)は、0から7までの8つの数字で1桁を数値を表現する

## 3ビットとの関係
8進数の1桁: 0～7で2進数を表現するとちょうど3ビット分表現できる

| 8進数 | 2進数 |
|-------|-------|
| 0     | 000   |
| 1     | 001   |
| 2     | 010   |
| 3     | 011   |
| 4     | 100   |
| 5     | 101   |
| 6     | 110   |
| 7     | 111   |

8進数: 644 を2進数へ
- 6: 110
- 4: 100
- 4: 100
これを並べる => `110 100 100`
