# 相関係数

2つのデータの関係性・関係度合いを数値で表したもの。

| 項目     | 説明               |
|----------|--------------------|
| 値の範囲 | -1～+1の間         |
| +1に近い | 正の強い相関       |
| -1に近い | 負の強い相関       |
| 0に近い  | 相関がほとんどない |

- 正の相関 : 身長高いと体重も重い(傾きがプラス)
- 負の相関 : 供給量が多いと価格も下がる(傾きがマイナス)
