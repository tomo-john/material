# セレクタ

CSSのセレクタは、スタイルを適用したいHTML要素を正確に`選択`するためのルール。

| 名称           | 書き方          | 説明                                                |
|----------------|-----------------|-----------------------------------------------------|
| タイプセレクタ | A { 〜 }        | 全てのA要素に適用。適用範囲が広く実務向けではない。 |
| 子孫セレクタ   | A B { 〜 }      | A要素に内包されるB要素の全てに適用。                |
| 子セレクタ     | A > B{ 〜 }     | A要素の直下の子要素であるB要素にだけ適用される。    |
| 擬似セレクタ   | A:hover { 〜 }  | 特定の状態のA要素だけに適用。(例ではマウスオーバー) |
| 隣接セレクタ   | A + B { 〜 }    | A要素の直後に隣接しているB要素にだけ適用。          |
| 属性セレクタ   | A[C] { 〜 }     | Cという属性を持ったA要素にだけ適用。                |
| classセレクタ  | .class名 { 〜 } | class名のclass属性を持った要素にだけ適用。          |
| idセレクタ     | #id名 { 〜 }    | id名というid属性を持った要素にだけ適用。            |

## スペースなしの場合

```
.class-a.class-b {
  ...
}
```

- 意味: AND条件
- 対象: `class-a`かつ`class-b`の両方のクラスを同時に持つ1つの要素
- HTML構造: 1つの要素に複数のクラスが指定されている
