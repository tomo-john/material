# $$

特殊変数の`$$`には現在のシェルのPIDが格納される。

その他:

- `$?` : 直線に実行されたコマンドの終了ステータス(exit値)
- `$n` : シェルスクリプトに渡されたn番目の引数(`$0`はスクリプト名)
- `$#` : シェルスクリプトに渡された引数の数

# read

標準入力、リダイレクションによるテキストファイルなどからデータを読み取り、それを変数に代入する。

readの後に変数を複数指定した場合、順に値が代入されていいく。

```
echo 'a b c d e' | while read x y z; do
  echo $z $x $y
done
```

- x : `a` が代入
- y : `b` が代入
- z : `c d e` が代入
- 実行結果は`c d e a b`
