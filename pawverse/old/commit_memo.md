# コミットメッセージ

| 接頭辞      | いつ？                             | 使用例                             |
|-------------|------------------------------------|------------------------------------|
| `feat:`     | 新機能を作った                     | `feat: ログイン機能の実装`         |
| `fix:`      | バグやエラーを直した               | `fix: リンク切れの修正`            |
| `docs:`     | ドキュメント(README等)の更新       | `docs: インストール手順を追記`     |
| `style:`    | 見た目、空白、セミコロンなど       | `style: インデントの調整`          |
| `refactor:` | コードの整理(機能は変更なし)       | `refactor: 重複処理の共通化`       |
| `perf:`     | パフォーマンス向上(動作を速くした) | `perf: 画像読み込みの高速化`       |
| `test:`     | テストコードを書いたり修正したり   | `test: ユーザー登録のテストを追加` |
| `chore:`    | 雑用(ライブラリ更新、設定変更など) | `chore: npmライブラリの更新`       |
