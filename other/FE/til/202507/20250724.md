# 確率

- ラインA : 全体の60%・不良が2%
- ラインB : 全体の40%・不良が1%

### それぞれの不良発生率を考える

- A : 0.6 x 0.02 = 0.12(1.2%)
- B : 0.4 x 0.01 = 0.04(0.4%)

### AとBを対比

A : B = 0.12 : 0.04 = 12 : 4 = 3 : 1

比率は `3 : 1` => 仮に不良品が4つ見つかったとき、Aで3つ、Bで1つのイメージ

1つ見つかった不良品がAであった確率は

3 / (1+4) = 3/4 => 0.75 (75%)

## 確率の加減法則

P(A OR B) = P(A) + P(B) - P(A AND B)

AもBも50%の場合 => 0.5 + 0.5 - (0.5 x 0.5) = 1 - 0.25 = 0.75 (75%)

AかB、もしくはABどちらもが発生する確率がこれ
