# usort

User defined Sortの略で、自分で定義したルールで並び替えができる。

### サンプル

```
<?php
// こんな連想配列があるとする
$todo_list = [
    ["id" => 2, "task" => "犬の散歩", "done" => false],
    ["id" => 3, "task" => "PHP学習", "done" => false],
    ["id" => 4, "task" => "ぽけいもん", "done" => false],
    ["id" => 1, "task" => "じょんに餌やり2", "done" => false] 
];

// $a と $b には、配列の中身が順番に入ってきて比較される
usort($todo_list, function ($a, $b) {
    // 「$aのid」と「$bのid」を比べてね、という命令
    return $a['id'] <=> $b['id'];
});
```

`function ($a, $b) {...}`が審判の役割をする。

PHPが配列の中から2つのデータを持ってきて、どっちが先頭かをこの関数に聞く。

## UFO演算子

`<=>`はPHP7以降に使用できる便利な演算子。

- 左が小さければ、マイナスの数値を返す(左を前に並べる)
- 右が小さければ、プラスの数値を返す(右を前に並べる)
- 左右が同じなら0を返す

この計算を1つの記号(演算子)でやってくれる。

昇順にしたいときはこれだけでOK。

もし降順にしたいときは、`function ($a, $b) {...}`の`{}`内の処理で、`return $b['id'] <=> $a['id']`とするだけ。
