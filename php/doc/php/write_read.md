# ファイル読み書き

PHPにはファイル操作用の関数がいくつかある。

| 操作     | 関数                  | 説明                                     |
|----------|-----------------------|------------------------------------------|
| 書き込み | `file_put_contents()` | ファイルに文字を書き込む(上書き or 追記) |
| 読み込み | `file_get_contents()` | ファイルの中身を一気に読み取る           |

## 例: memo.txtに書き込み・読み込み

```
<?php
$file = 'memo.txt'; // 保存するファイル名

// 書き込み(ファイルがなければ自動で作られる)
file_put_contents($file, "こんにちは！PHPで書き込みました🐶\n");

// 読み込み
$content = file_get_contents($file);

echo 'ファイルの中身はこちら🐶<br>';
echo nl2br($content); // 改行を<br>に変換して表示
```

### 補足

- `file_put_contents()`

  => ファイルがなければ新規作成・あれば上書き

  => 追記したい場合は第三引数に`FILE_APPEND`を指定

- `file_get_contents()`

  => ファイル全体を`文字列`として1発で読込む
