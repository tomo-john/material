# SSDにUbuntu

## 概要
外付けSSDにUbuntu OSをインストール。

既存PCの元々あるWindows環境を汚さない。

=> ブートメニューからWindowsかUbuntuかを選ぶ

=> ディアルブートにはしない

- Ubuntuのisoをダウンロード
- インストール用USBの作成
- 外付けSSDにUbuntuをインストール

### Ubuntuのisoをダウンロード

公式サイトからダウンロードを実施。

今回は、`ubuntu-24.04.4-desktop-amd64.iso`

### インストール用USBの作成

16GBのUSBを今回は用意。

Rufusをダウンロード(`rufus-4.14.exe`)

Rufus起動:
- デバイス: 16GBのUSBメモリ
- ブートの種類: 選択 => isoイメージファイルを選ぶ
- パーティション構成: GPT
- その他は規定値
- スタート: ISOイメージモード(推奨)を選択

=> 20分程で完了

> ここで作成したUSBは今後使いまわせる

### 外付けSDDにUbuntuをインストール

1TBの外付けSSDを今回は用意。

PCにSSD, USBを取り付けた状態で起動 => ブートメニューへ(DellならF12)

起動メニューでUSBメモリを選択(Ubuntuインストーラー起動)

起動選択画面でしばらく放置してると勝手にUbuntuのtry版が起動した。

=> 起動後にインストールできるので進む

パーティション:

- 1. sdaの中のsda1 NTFS 空白 1.00TBを一度 `-` で削除

    => sda には空き容量 1.00TBだけになる

- 2. 次に左したのブートローダのデバイス選択で外付けSSDが選択できるので選択

- 3. すると勝手に `sda1 FAT32 /boot/fdi 1.13GB`が自動で作られた

    => 空き容量 999.07GBがある

4. 空き容量に + で 残り容量全部 `Ext4` `/` で作成

最終形:
```
sda1  FAT32  /boot/efi  1.13GB
sda2  Ext4   /          999.07GB
# => 次へが押せる🐶
```
