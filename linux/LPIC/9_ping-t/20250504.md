# for

```
for INT in 1 2 3; do
  echo $INT
done
```

# 特殊変数

- `$?` : 直前に実行されたコマンドの終了ステータス(exit値)
- `$n` : シェルスクリプトに渡されたn番目の引数(`$0`はシェルスクリプト名、101個目以降の引数は`${10}`, `${11}`...)
- `$$` : 現在のシェルのPID
- `$#` : 実行時に指定された引数の数

# alias

```
alias rm='rm -f'
```

# exec

指定したコマンドを既存のジョブ(シェルのプロセス)と置き換えて実行するコマンド。

通常はコマンド実行時、シェルのプロセスとは別に、実行するコマンドのプロセスが生成される。
