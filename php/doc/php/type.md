# 型

| 型名     | 意味                     | 例                                     |
|----------|--------------------------|----------------------------------------|
| `string` | 文字列                   | ユーザー名、メールアドレス、HTML文字列 |
| `int`    | 整数                     | ID、年齢、カウント                     |
| `floot`  | 浮動小数点               | 価格、計算結果、小数点以下の値         |
| `bool`   | 真偽値(`true`, `false`)  | フラグ、判定                           |
| `array`  | 配列                     | 複数データ、連想配列                   |
| `object` | クラスのインスタンス     | `PDO`や自作した`Dog`クラスなど         |
| `?Type`  | Nullable型               | `?string`で文字列または`null`を許可    |
| `void`   | 何も返さない             | 値を返さないメソッド                   |

## 戻り値の型宣言

その関数(メソッド)がどんな種類のデータを返すのかを明確に宣言するもの。

```
static function getPdoConnection(): PDO {
  ...
  return $pdo // $pdo はPDOクラスのオブジェクト
}
```

この`: PDO`とすることで、このメソッドは必ず`PDO`クラスのインスタンス(オブジェクト)を返しますとPHP自身の宣言している。

もし、このメソッドが`PDO`でないもの(`null`やただの`string`など)を返そうとするとPHPがエラーを出してくれる。
