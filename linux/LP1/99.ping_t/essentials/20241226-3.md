# unzip

zip形式の圧縮ファイルを展開する。

| 圧縮  | 展開    | 形式    |
|-------|---------|---------|
| gzip  | gunzip  | gz形式  |
| bzip2 | bunzip2 | bz2形式 |
| xz    | unxz    | xz形式  |
| zip   | unzip   | zip形式 |

# プライベートブラウジング

プライベートブラウズ(プライベートブラウジング)は、閲覧したwebサイトの履歴やパスワード、cookieなどを自動的に破棄することで、

ブラウザに情報を保持しておかないようにするしくみ。

例: Mozilla Firefoxでは以下の情報が保存されない。

- Webページの閲覧履歴や入力履歴
- パスワード(自動入力されない)
- cookie
- キャッシュ

これらはブラウザがその利用を終えたタイミングで削除される。
