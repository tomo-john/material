# 指数

| 指数計算            | 例               |
|---------------------|------------------|
| `a^m * a^n = a^m+n` | 2^4 * 2^3 = 2^7  |
| `a^m / a^n = a^m-1` | 2^4 / 2^3 = 2^1  |
| `(a^m)^2 = a^m*n`   | (2^4)^3 = 2^12   |
| `a^0 = 1`           | 2^0 = 1, 8^0 = 1 |
| `a^-m = 1/a^m`      | 2^-4 = 1/2^4     |
`
