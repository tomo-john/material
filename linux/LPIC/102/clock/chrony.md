# Chrony

Chrony(クローニー)はntpd/ntpdateの代替えとなるNTPサーバー/クライアントソフトウェア。

- `chronyd` : デーモンプロセス
- `chronyc` : クライアントコマンド

この2つから構成されている。

=> 設定ファイルは`/etc/chrony.conf`

---

# chronyc

chronydの管理を行うコマンド。

```
chronyc [サブコマンド]
```

| サブコマンド | 説明                                         |
|--------------|----------------------------------------------|
| activity     | NTPサーバーのオンライン/オフラインの数を表示 |
| sources      | 時刻ソースの情報を表示                       |
| sourcestats  | 時刻ソースの統計情報を表示                   |
| tracking     | トラッキングを確認                           |
| quit         | 対話状態を終了する                           |

=> サブコマンドなしで`chronyc`を実行すると対話モードになる
