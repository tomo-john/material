# null合体演算子

例: `$errors = $_GET['errors'] ?? [];`

もし左が存在していて、`null`でなければそれを使う、なければ右側を使うという意味。

`isset()`よりもスッキリしていいてPHP7以降ではよく使用される。

### isset()とは？

変数が宣言されており、かつ`null`でない場合に`true`を返す関数。

- 未定義の変数に使ってもエラーはでない
- `null`値の変数に使うと`false`を返す
- `0`, `""(空文字)`, `false`に対しては`true`(存在する)と判定する

```
# isset()
$name = isset($username) ? $username : 'Guest';

# null合体演算子
$name = $username ?? 'Guest';
```

### empty()との違い

`empty()`は、`isset()`よりも厳しくチェックする。

変数が`null`, `0`, `""(空文字)`, `[](空の配列)`, `false`などの場合に`true`(空である)を返す。

`isset()`は`null`または`未定義`の場合にのみ`false`を返す。

# 三項演算子

単純な`if ... else ...`文を1行で簡潔に書くための構文。

基本構文:

```
(条件式) ? (条件式がtrueの場合に返す値) : (条件式がfalseの場合に返す値);
```

- `?`と`:`の組み合わせで使用する
- もし条件式が真ならAを、偽ならBをという意味になる

例:

```
$num = 20;
$result = ($num >= 20) ? 'OK' : 'NG';
```
