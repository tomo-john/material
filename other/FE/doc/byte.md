# データの単位

- 1bitで0と1の2通りを表す
- `1byte = 8bit`なので、1byteで2の8乗通り(=256通り)を表すことが可能

| 単位 | 記号 | 累乗  |
|------|------|-------|
| キロ | k    | 10^3  |
| メガ | M    | 10^6  |
| ギガ | G    | 10^9  |
| テラ | T    | 10^12 |


| 単位     | 記号 | 累乗                   |
|----------|------|------------------------|
| ミリ     | m    | 10^-3(0.001)           |
| マイクロ | μ    | 10^-6(0.000001)        |
| ナノ     | n    | 10^-9(0.000000001)     |
| ピコ     | p    | 10^-12(0.000000000001) |
