# データの単位

- 1bitで0と1の2通りを表す
- `1byte = 8bit`なので、1byteで2の8乗通り(=256通り)を表すことが可能

# 命令実行速度＆単位変換ミス防止表(FE試験用)

## 1. 基本の計算式

- 命令数 [命令 / 秒]
- 命令数/秒 = クロック周波数[Hz] / CPI
- Hz(ヘルツ)＝ 1秒あたりのクロック数
- CPI(Cycles Per Instruction) ＝ 1命令あたりに必要なクロック数の平均
- MIPS(Million Instructions Per Second)
- MIPS = (命令数/秒) / 1,000,000(10^6)
