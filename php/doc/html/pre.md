# brとpreタグ

| タグ    | 役割                       | 使いどころ                                   |
|---------|----------------------------|----------------------------------------------|
| `<br>`  | 改行                       | テキストの途中などに1行だけ改行を入れたい    |
| `<pre>` | この中の文字をそのまま表示 | 文章の改行・空白・インデントをそのまま見たい |

## brタグ

- HTMLの改行タグ
- 閉じタグなし(`<br>`または`<br />`)
- ここで1個改行してくださいという意味

## preタグ

- preformatted text : 整形済みテキスト用のタグ
- HTML上では無視されるはずの連続スペース、タブ、改行などをそのまま画面に表示
- 範囲指定タイプのタグなので閉じタグ(`</pre>`)が必要

## PHPでよく使うテクニック

1. PHPの改行(`\n`)を`<br>`に変換

```
echo nl2br("hello\nDog");
```

2. ユーザー入力を安全に`<pre>`で表示

```
echo "<pre> . htmlspecialchars($text, ENT_QUOTES) . </pre>";
```
