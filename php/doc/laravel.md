# Laravle

## 環境構築

### 必要なもの

- Homebrew(Mac OS)
- Node.js・npm
- PHP
- Composer

### Homebrew

macOs用のパッケージ管理システム。

今回必要なPHP, Composer, Node.js, MySQLなの必要ツールを管理するために使用。

LinuxでいうところのYum/DNF(Red Hat系)、apt/dpkg(Debian系)みたいなもの。

### Node.jsとnpm

Node.jsとはサーバーサイドでJavaScriptを動かすための実行環境のこと。

npmは`Node Package Manager`の略で、Node.jsのためのパッケージ管理システム。

npmはJavaScriptのライブラリやツールを管理するために使われる。

```
# Node.jsとnpmをまとめてインストール
brew install node
```

### PHP

PHP: PHP Hypertext Preprocessor

サーバーサイド・スクリプト言語。

### Composer

PHPにおける依存関係管理ツール(パッケージマネージャー)

ComposerはLaravelを使う上で必須。
