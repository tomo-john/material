# NAND

| A | B | X |
|---|---|---|
| 0 | 0 | 1 |
| 1 | 0 | 1 |
| 0 | 1 | 1 |
| 1 | 1 | 0 |

否定論理積(2入力NAND)は、2つの入力がともに1のときにだけ結果が0、その他の場合は1となる論理演算。

NAND素子を組み合わせることによってAND, OR, XOR, NOT, NORなどの回路を構成することができる。

多入力NANDは入力が`1つでも0なら1を返す`論理回路。

=> 4入力NANDなら4つの入力のいずれかが0であるときに1を出力する、または全ての入力が1のときに0を出力する論理回路
