# 繰り返し処理

| 構文         | 説明                       | よく使う場面             |
|--------------|----------------------------|--------------------------|
| for          | 回数が決まっている繰り返し | 数値のカウントアップなど |
| while        | 条件が真(true)の間繰り返す | 入力チェックなど         |
| do ... while | 最低1回は実行される`while` | メニューの繰り返しなど   |
| foreach      | 配列や連想配列を順に処理   | 配列の要素を全部出力     |

## for

```
<?php
for ($i = 1; $i <= 5; $i++) {
  echo "わんこが{$i}回目に吠えた！</br>";
}
```

- `$i = 1` : カウンタ初期値
- `$i <= 5` : 繰り返し条件
- `$i++` : 1ずつ増える(インクリメント)

## while

```
<?php
$i = 1;
while ($i <= 3) {
  echo "わんこがおやつを{$i}個食べた</br>";
  $i++;
}
```

- 条件が`true`の間だけ繰り返す
- `$i++`を忘れると無限ループになるので注意
- 条件によっては一度も実行されないこともある

## do ... while

```
<?php
$i = 1;
do {
  echo "{$i}匹目のわんこ</br>";
  $i++;
} while ($i <=10);
```

- 最初の1回は必ず実行される

## foreach

```
<?php
$dogs = ["ダックス", "ゴールデン", "ラブラドール"];

foreach ($dogs as $dog) {
  echo "{$dog}が走った</br>";
}
```

- 配列の要素をひとつずつ順番に取り出して使う
- `$dog`には配列の中身(値)が順に格納される

## 連想配列

```
$words = [
  "いぬ" => "dog",
  "うさぎ" => "rabbit",
  "くま" => "bear"
];

foreach ($words as $key => $val) {
  echo "{$key}は英語で{$val}です。</br>";
}
```

- 連想配列のキーが`$key`に、値が`$val`に格納

## break

繰り返し処理を途中でやめる命令。

`if`文などと組み合わせて使用する。

`break 2;`のように数字を指定すると、2階層分のループを一気に抜けることもできる。

```
<?php
for ($i = 0; $i <=100; $i++) {
  echo "{$i}回目の処理です。";
  echo "{$i}の2乗は50未満です。</br>";
  if ($i * $i >= 50) {
    echo "{$i}の2乗で50以上になりました。</br>";
    break;
  }
}
```

## continue

この1回だけは飛ばして、次のループへ進むという命令。

```
<?php
for ($i = 1; $i <= 10; $i++) {
  if ($i % 3 == 0) {
    echo "この回は走りません</br>";
    continue;
  }
  echo "じょんの{$i}回目のダッシュ</br>";
}
```
