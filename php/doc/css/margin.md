# 余白

- `padding` : 内側の余白
- `margin` : 外側の余白

## 効率的な書き方

| 場面                   | 書き方                      | 補足                               |
|------------------------|-----------------------------|------------------------------------|
| 上下左右の値が同じ     | `margin: 10px;`             | 上下左右ともに10px                 |
| 上下と左右で値が同じ   | `margin: 10px 5px;`         | 上下が10px、左右が5px              |
| 左右が同じで上下が違う | `margin: 10px 5px 2px;`     | 上が10px、左右が5px、下が2px       |
| 上下左右それぞれ       | `margin: 10px 8px 2px 5px;` | 左から上・右・下・左の順(時計回り) |

=> `padding`でも書き方は同じ

## box-sizing

最初の状態ではpaddingとmarginは横幅や高さに含まれない。

=> 横幅を100pxと指定しても、paddingやmarginを含めると100pxより大きくなる

`box-sizing`プロパティの値を`border-box`にすると、横幅や高さにpaddingとmarginを含めることが可能。

## 中央寄せ(ブロックボックス)

`width`プロパティで横幅を指定した要素のmargin-left, margin-rightをそれぞれ`auto`に指定する。

左右の余白が均等になり、中央に配置される。

中央寄せは左右に余白がないとできない。

インラインボックスを中央寄せにするには、その要素が含まれている親のブロックボックスに`text-align:center;`を記載する。
