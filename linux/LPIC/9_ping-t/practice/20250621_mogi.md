# id

- -u : UIDを表示
- -g : プライマリグループのGIDを表示
- -G : 所属するすべてのグループのGIDを表示

# /etc/X11/xdm

- xdm-config : XDMの設定ファイル
- Xresources : XDMログイン画面のデザイン設定
- Xaccess : ホストからXDMへアクセス許可の設定
- Xsetup_0 : XDMログイン画面表示前に実行されるスクリプト
- Xsession : XDMのログイン後に実行されるスクリプト
