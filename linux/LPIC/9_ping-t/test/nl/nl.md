# nl

指定したファイルに行番号をつけて出力する。

| nl      | cat    | 動作                                     |
|---------|--------|------------------------------------------|
| nl -b a | cat -n | 空行を`含めた`すべての行に行番号をつける |
| nl -b t | cat -b | 空行を`除いた`すべての行に行番号をつける |

=> nlにオプションなし(nlだけ実行した)場合は、`nl -b t`と同じ動き(デフォルト)

=> デフォルトでは空行に行番号をつけない
