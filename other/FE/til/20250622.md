# サンプルプログラム

```
#!/bin/bash

age=$1
ret=0

if [ "$age" -le 3 ]; then
  ret=100
elif [ "$age" -le 9 ]; then
  ret=300
else
  ret=500
fi

echo "$ret"
```

- ageが0から3の場合、retは100
- ageが4から9の場合、retは300
- ageが10以上の場合、retは500
