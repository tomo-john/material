# img

画像をWebページに表示するためのタグ。

基本的に画像を読み込んで表示するだけのシンプルなタグだがよく使われる。

## 基本構文

```
<img src="画像のURL" alt="代替テキスト">
```

## 属性

| 属性名  | 説明                                                   | 例                                 |
|---------| -------------------------------------------------------|------------------------------------|
| src     | 画像ファイルのURLやパスを指定します                    | `src="images/photo.jpg"`           |
| alt     | 画像が表示できないときに代わりに表示されるテキスト     | `alt="風景の写真"`                 |
| width   | 画像の幅をピクセルやパーセンテージで指定               | `width="300"` または `width="50%"` |
| height  | 画像の高さをピクセルやパーセンテージで指定             | `height="200"`                     |
| title   | 画像の説明。マウスを乗せたときにツールチップとして表示 | `title="夕日の写真"`               |
| loading | 画像の読み込み方法を指定します(HTML5)                  | `loading="lazy"`(遅延読み込み)     |
