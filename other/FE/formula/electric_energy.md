# 電力量

電力量はある時間に、どれだけの電気エネルギーを使用したかを表す量。

| 項目   | 単位         | 意味                            |
|--------|--------------|---------------------------------|
| 電力量 | Wh(ワット時) | 使ったエネルギーの量            |
| 電力   | W(ワット)    | 1秒間に使う電気の量(出力の強さ) |
| 時間   | h(時間)      | 電力を使った時間                |

```
電力量(Wh) = 電力(W) * 時間(h)
```

# 電力

電力は1秒あたりに消費する電気エネルギー。

```
電力(W) = 電流(A) * 電圧(V)
```
