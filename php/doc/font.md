# フォント関連

| プロパティ   | 役割             |
|--------------|------------------|
| font-family: | フォントの種類   |
| font-size:   | フォントのサイズ |

## font-family

値にはフォント名が入る。

`,(カンマ)`で区切って、複数のフォントを指定できる。

```
font-family: 'フォントファミリ名1','フォントファミリ名2',総称ファミリ名
```

- フォントファミリ名は`,(カンマ)`で区切り複数指定できる
- このとき、フォントファミリ名1が優先度1、フォントファイミ名2が優先度2
- フォントファミリ名は総称ファミリ名と区別するため、`'`, `"`で囲む
- 総称ファミリ名は囲まない
- 総称ファミリ名は、指定されたフォントがどれも存在しなかった場合の代替えフォント

## font-size

値にはpxなどの単位を伴う数値やlargeなどの大きさを表すキーワードが入る。
