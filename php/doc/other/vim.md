# vim

## cit

- c: Change(消去してインサートモードに入る)
- i: Inner(内側)
- t: Tag(タグ)

=> 今カーソルがあるタブの中身(テキスト)を全部消して、書き直す

### 他の仲間たち

| コマンド | 意味               | 動作                               |
|----------|--------------------|------------------------------------|
| `cit`    | Change Inner Tag   | タグの中身だけを書き換える         |
| `cat`    | Change Around Tag  | タグ全体を書き換える               |
| `ci"`    | Change Inner Quote | `""`の中身(属性値など)を書き換える |
| `dit`    | Delete Inner Tag   | タグの中身を消すだけ               |
