# Tailwindのサイズ指定(Tailwindスケール)

## 数値ベースのスケール

`width`, `padding`, `margin`などの具体的な間隔を指定するときに使用される。

- `1` = 0.25rem(4px)
- `4` = 1rem(16px)
- `8` = 2rem(32px)

=> 4の倍数で段階的に増加する

### 使用例

- `w-16`: `width: 64px`
- `h-32`: `height: 128px`
- `p-4` : `padding: 16px`
- `m-8` : `margin: 32px`

## キーワードベースのスケール

特定のデザインやレイアウトの意図に合わせて、数値ではなくキーワードが使用される。

### max-w-

コンテンツの最大幅を制限するために使われるキーワード。

| Tailwind  | 最大幅 (Rem) | 最大幅 (Pixel) | 目的                         |
|-----------|--------------|----------------|------------------------------|
| max-w-xs  | 20rem        | 320px          | 非常に狭い(モバイル幅など)   |
| max-w-md  | 28rem        | 448px          | 標準的なカード・フォームの幅 |
| max-w-lg  | 32rem        | 512px          | 少し広めのコンテンツ         |
| max-w-7xl | 80rem        | 1280px         | デスクトップで非常に広い幅   |

### 特殊なサイズ

| Tailwind | CSS相当        | 役割                               |
|----------|----------------|------------------------------------|
| w-full   | width: 100%;   | 親要素の幅いっぱい                 |
| w-screen | width: 100vw;  | ビューポート(画面)の幅いっぱい     |
| h-screen | height: 100vh; | ビューポート(画面)の高さいっぱい   |
| w-auto   | width: auto;   | ブラウザのデフォルトの動作に任せる |
