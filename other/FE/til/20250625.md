# 基数変換

10進数 => N進数の場合:

- 整数部は、Nで割って下から余りを並べる
- 小数部は、Nを掛けて順に整数部を並べる

```
# 10進数の6を2進数
6 / 2 = 3 ... 0
3 / 2 = 1 ... 1
1 / 2 = 0 ... 1

=> 10進数の6は2進数では110

# 10進数の0.375を2進数
0.375 * 2 = 0.75 # 0
0.75  * 2 = 1.5  # 1
0.5   * 2 = 1.0  # 1 (小数部が0になったら終了)

=> 10進数の0.375の2進数は0.011

10進数の6.375は2進数では110.011になる
```

```
# 10進数13.375
2^3 = 8 # 13.375 > 8 => 1
2^2 = 4 # 5.375  > 4 => 1
2^1 = 2 # 1.375  < 2 => 0
2^0 = 1 # 1.375  > 1 => 1
.
2^-1 = 0.5   # 0.375 < 0.5  => 0
2^-2 = 0.25  # 0.375 > 0.25 => 1
2^-3 = 0.125 # 0.125 = 0.15 => 1

# => 2進数では1101.011
```

2進数から8進数・16進数への基数変換は、小数点を基準に8進数なら3桁、16進数なら4桁ずつに区切って計算する。

```
# 2進数: 110101111.101
# 3桁区切り: 110 101 111.101
# 4桁区切り: 0001 1010 1111.1010

8進数  => 657.5
16進数 => 1AF.A
```
