# nl

行番号をつけて表示するコマンド。

デフォルトでは`空行には行番号をつけない` => `cat -b`と同じ

行番号をつけるだけでなく、簡単な整形も可能。

## オプション

- -b a : 空行にも行番号をつける(`cat -n`と同じ)
- -b t : 空行を除いたすべての行に行番号をつける(`デフォルト`)
- -n ln : 行番号を左詰めで表示
- -n rn : 行番号を右詰めで表示(`デフォルト`)
- -n rz : 行番号を右詰め + 0埋めで表示
- -s 文字列 : 行番号の後に入れる区切り文字列を変更(デフォルトはタブ)
- -w 桁数 : 行番号の桁数(何桁で表示するか)

## 使用例

```
# 行番号4桁の右詰め、区切り文字が「:」
nl -n rz -w 4 -s ":" sample.txt
```
