# CSS

CSSを読み込む3つの方法。

## 外部ファイル読み込み

`.css`ファイルを作って、`<link>`で読み込む方法。

`<head>`タグ内に`<link ref="stylesheet" href="xxx.css">`

```
<head>
  <link rel="stylesheet" href="style.css">
</head>
```

複数のCSSファイルを指定した場合は、上の行から順番に読まれる。

=> `reset.css`をあてる場合は上に書く

## styleタグ

HTMLファイルの中に直接CSSの場所を作る方法。

これも`<head>`タグ内に記述する。

```
<head>
  <style>
    h1 { color: red; }
    p { color: blue; }
  </style>
</head>
```

このHTMLファイルにだけ適用されるCSS。

## インラインスタイル(直書く)

タグの中に直接書き込む方法。

優先度が最強だけど、実務向きではない。(検証・お遊び用)

```
<h1 style="color: red;">ここだけ赤くなる</h1>
```

# classとid

| 特徴       | class                    | id                       |
|------------|--------------------------|--------------------------|
| CSS書き方  | `.`                      | `#`                      |
| HTMLでの数 | 1ページに何度でも使える  | 1ページにつき1回のみ     |
| 優先度     | ふつう                   | 強い(classより優先)      |
| 使用例     | 同じデザインのボタンなど | ページの大枠・JavaScript |

## 名前の付け方

classもidも任意の名称をつけることができるが、いずれも先頭の文字を数字から始めることはできない。

2単語をつなぐ場合は、以下の代表的な付け方をすると誰がよんでもわかりやすい :dog:

- キャメルケース: `superDog`
- スネークケース: `super_dog`
- ケバブケース: `super-dog`
