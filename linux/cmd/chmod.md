### chmod
change mode
ファイルやディレクトリの権限を変更する。
- 4 => read : 読み取り
- 2 => write : 書き込み
- 1 => execute : 実行

各桁の数値はこれらの値の合計

```
# ls -l した時に左側に出るやつ
-rwxrw-r--
# (左から)
-   : ファイルかディレクトリか(ディレクトリならd)
rwx : ユーザーの権限(これは7)
rw- : グループの権限(これは6)
r-- : その他ユーザーの権限(これは4)
# 例
chmod 764 'filename'
=> -rwxrw-r-- になる
```
